Automate Posting to Mastodon via Web Requests

One of my favorite applications that I use for automation is If This Then That aka IFTTT. I have been using it forever and it allows you to automate a plethora of things when something happens. For example, maybe you want ot create a spreadsheet of every favorited tweet you have, or maybe save them to a list somewhere, or maybe automatically turn on your lights at sunset. All of this is possible with IFTTT and is crazy easy to use with connectors.
